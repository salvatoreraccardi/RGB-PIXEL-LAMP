# RGB PIXEL LAMP
A simple code for control a rgb led pixel ring.

**Main functions**
- [x] RGB SYNC EFFECT
- [x] Basic function
- [ ] Light dependent system
- [ ] Bluetooth control
- [ ] Temperature / humidity reading
```sh
#include <Adafruit_NeoPixel.h>
```

```sh
Adafruit_NeoPixel pixels = Adafruit_NeoPixel(LED, OUTPIN, NEO_GRB + NEO_KHZ800);
```
